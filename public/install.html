Wake on LAN — two steps required:

1. Enable in BIOS/UEFI: Advanced → APM Configuration → Power On By PCI-E: Enabled and ErP Ready: Disabled.

2. Enable on the NIC (run while machine is online, replace eno1 with your interface):
ip link show — to find your interface name
sudo ethtool -s eno1 wol g — to enable WoL

To make it persist across reboots:
sudo systemctl enable wol.service after the agent install creates it automatically.
